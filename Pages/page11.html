Python Script Guide

Overview

This page provides a guide to the python scripts I developed for scraping and analyzing Usenet data from the Google Groups archive. I chose the Google Groups archive because it offers a comprehensive and accessible repository of historical Usenet content, making it an ideal resource for examining how early online communities engaged in discussions. The scripts allow users to extract detailed data on thread titles, comments, and metadata, enabling in-depth analysis of conversations within a newsgroup.

Why Google Groups?

Google Groups serves as a centralized platform that preserves extensive Usenet archives in a consistent format, making it easier to access and analyze compared to other archives. Its structured data allows for systematic scraping, which is crucial for large-scale research projects. By using Google Groups, we can tap into a rich dataset of Usenet discussions spanning decades, offering valuable insights into the early days of online communication.

Whatâ€™s Available in the Scripts?

The following Python scripts are designed to be used together for a comprehensive data collection process:

  • thread_scraper.py This script is designed to gather thread titles and metadata from a specified Usenet newsgroup on Google Groups. It collects key information such as thread titles, creation dates, authors, and the number of replies. This provides an overview of the newsgroup's activity, helping to identify key discussions and topics.
  • comment_scraper.py Building on the output of thread_scraper.py, this script scrapes all comments and their metadata from each thread listed. It extracts details such as the comment text, author, timestamp, and any additional metadata, allowing for a deeper analysis of the conversation dynamics within each thread.

By running these scripts sequentially, you can extract a complete dataset from a newsgroup, from thread-level summaries to detailed comment-level data.

Set Up Your Environment

Ensure Python 3.7 or later is installed on your system. Install the necessary libraries using pip:

pip install -r requirements.txt

Download the scripts thread_scraper.py and comment_scraper.py to your preferred working directory.

Run thread_scraper.py

Open the script in a text editor and specify the URL of the Usenet newsgroup you want to scrape. Execute the script using your command line or Python IDE:

python thread_scraper.py

This script will output a CSV file with thread titles and metadata.

Run comment_scraper.py

Use the CSV output from thread_scraper.py as input for this script. Edit the script to reference the CSV file location. Run the script:

python comment_scraper.py

The result will be a CSV file containing all comments and associated metadata from each thread in the specified newsgroup.

Application of Scripts in My Research

My research focuses on Usenet before the Great Renaming, specifically targeting some of the most popular newsgroups as identified in the Usenet Readership Summary Report for September 1986. I have concentrated on the newsgroups net.motss, net.singles, net.politics, net.religion, net.news, and net.med because they are particularly relevant to my research on AIDS discussions and related social issues.

Using these scripts, I have extracted extensive data from these groups, which has enabled a detailed examination of the early discussions on AIDS and homosexuality. For my project, I've performed additional cleaning to filter out threads that don't discuss AIDS or homosexuality. However, for this webpage, I chose to omit keyword filtering to provide a more comprehensive overview of early Usenet discussions, which I believe will be valuable to a broader audience interested in this historical period.
